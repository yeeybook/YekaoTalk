# YekaoTalk
This is a basic messenger app for Android based on Firebase (Realtime Database) 

## Features
* Authentication (Email)
* Profile Picture
* Status
* Text Message
* Unread Message Count
* 1:1 Chat 
* Group Chat
* Push Notification
* Last Message
* Time Stamp
